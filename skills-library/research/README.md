# research drawer

Universe definition, idea sourcing, competitive read, and the thesis spine.

| Skill | Kind | Does |
|---|---|---|
| universe-screener | deterministic | Filter US public companies by SIC / market-cap / liquidity. |
| catalyst-flagger | hybrid | Tag event-driven catalysts from recent 8-Ks + news. |
| moat-analyzer | hybrid | Assess competitive advantage from the 10-K, backed by Python margins. |
| thesis-recorder | deterministic | Persist a thesis + KPI watchlist (read later by kpi-tracker). |

Each skill is documented by its own `SKILL.md`. See the [library README](../README.md).
