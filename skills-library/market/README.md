# market drawer

Market and company data retrieval (free/keyless sources, cached to SQLite).

| Skill | Kind | Does |
|---|---|---|
| price-fetcher | deterministic | Price history + return + annualized volatility + drawdown. |
| news-fetcher | deterministic | Recent headlines + SEC filing alerts (RSS). |
| fundamentals-fetcher | deterministic | Structured financials from SEC XBRL facts. |

Each skill is documented by its own `SKILL.md`.
